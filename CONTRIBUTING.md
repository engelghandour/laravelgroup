# Contributing to Laravel Group Project

Welcome to the Laravel Group Project for ICT Academy! This guide will help you contribute effectively to our collaborative learning environment.

## Project Overview

This project is designed for ICT Academy students to learn Laravel development under the supervision of **Eng. Mohamed Mansour**.

## Getting Started

1. **Fork the repository** to your GitHub account
2. **Clone your fork** locally
3. **Create a new branch** for your feature or assignment
4. **Make your changes** following the guidelines below
5. **Test your changes** thoroughly
6. **Submit a pull request** for review

## Coding Standards

### PHP Standards
- Follow PSR-4 autoloading standards
- Use meaningful variable and function names
- Add comments for complex logic
- Follow Laravel naming conventions

### Git Workflow
- Use descriptive commit messages
- Keep commits small and focused
- Branch naming: `feature/your-name-feature-description`
- Example: `feature/ahmed-user-authentication`

## Learning Objectives

Each contribution should demonstrate understanding of:
- MVC Architecture
- Laravel routing
- Eloquent ORM
- Blade templating
- Database migrations
- Form validation
- Authentication (where applicable)

## Code Review Process

1. All code must be reviewed by peers
2. Final approval by **Eng. Mohamed Mansour**
3. Address feedback constructively
4. Learn from others' code and suggestions
